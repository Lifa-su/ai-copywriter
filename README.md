## ✨ Features / 功能亮点

- 📝 **8 Content Types** — Xiaohongshu posts, WeChat Moments, product descriptions, business emails, self-introductions, weekly reports, apology letters, thank-you notes
- 🎨 **Tailored Forms** — Each type has dedicated input fields to guide you through key details
- 🎭 **4 Tone Styles** — Formal / Casual / Humorous / Professional — switch freely
- 🔢 **Multi-Version Output** — Generate 1–5 versions at once, pick your favorite
- 📋 **One-Click Copy** — Instantly copy to clipboard
- 💾 **History Records** — Auto-saved locally, browse and delete anytime
- 📱 **Responsive Design** — Works on mobile, tablet, and desktop
- 🚀 **Zero Build, Zero Backend** — Pure frontend, single HTML file, just open and use

## 🚀 Quick Start / 快速开始

### Option 1: Online Demo（推荐）

👉 Visit **[https://lifa-su.github.io/ai-copywriter](https://lifa-su.github.io/ai-copywriter)**

### Option 2: Run Locally

```bash
# Clone the repo
git clone https://github.com/Lifa-su/ai-copywriter.git

# Open in browser — that's it!
open ai-copywriter/index.html
```

### Setup / 配置

1. Get your API Key from [DeepSeek Platform](https://platform.deepseek.com/api_keys)
2. Paste it into the app and click **Save**
3. Choose a content type → Fill in details → Select tone → Click **Generate** ✨

## 🛠 Tech Stack / 技术栈

| Layer | Technology |
|-------|-----------|
| Frontend | HTML + CSS + JavaScript (single file) |
| AI Model | [DeepSeek](https://platform.deepseek.com/) Chat API (`deepseek-chat`) |
| Storage | `localStorage` (API Key + history) |
| Hosting | GitHub Pages |
| CI/CD | GitHub Actions |

> No build tools. No backend. No dependencies. Just one `index.html`.
